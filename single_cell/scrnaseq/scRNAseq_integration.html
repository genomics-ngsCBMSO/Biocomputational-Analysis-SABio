Step 1. Preprocessing

Before applying deconvolution, both datasets must be made comparable. The single-cell reference dataset must be filter and properly processed. Additionally, the bulk RNA-seq data must be preprocessed to ensure compatibility with the single-cell reference. Besides the basic scRNAseq analysis, we perform the following steps:

  • Remove mixed and undefined cells
  • Gather all the annotated cell types
  • Filter out rare cell types
  • Identify strongly associated genes to each cell type (cell identity markers)
  • Apply transformations (log-CPM, log-TPM) to place datasets on comparable scales

Step 2: Building the reference (signature matrix)

The single-cell dataset must be summarized into representative expression profiles for each cell type. This reference is known as the signature matrix and is critical for deconvolution accuracy.
